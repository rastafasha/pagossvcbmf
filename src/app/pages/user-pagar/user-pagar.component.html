Como pagar?

Dirijase a la app o web del metodo preferido,

  1. Copie los datos suministrados para procesar el pago:
    Correo: correopago@gmail.com,
    Metodo: Transferencia,
    Monto: $USD 40.00,

  2. Dirijase a la app o web del metodo preferido
  3. Ingrese los datos correctamente, y verifique antes de pagar
  4. Copiar el numero de referencia del pago o transferencia
  5. Pegar el numero de la referencia en el campo correspondido
  6. Adjuntar la foto del pago
  7. Espere el cambio de status.
